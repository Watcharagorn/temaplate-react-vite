### Template Footprint
```bash
# initial vite-react
yarn create vite template-app --template react-ts

# initial tailwind
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

### How to run
```bash
# run app local
npm run dev

# production
npm run build
npm run preview
```
> Server running on =>  http://localhost:8080/
